## 2. `char` (Character)

* Stores a **single character** in single quotes `' '`.
* Internally stored as an **ASCII code** (numeric).


### Size & Range:

* **1 byte**
* Range: **-128 to 127** (signed char) or **0 to 255** (unsigned char)

### ASCII Example:

