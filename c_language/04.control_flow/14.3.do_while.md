# 14.3: The `do-while` Loop in C

---

## 1) What is a `do-while` loop?

A **`do-while` loop** is similar to a `while` loop, but with one big difference:
It **executes the loop body first, then checks the condition**.

This means the `do-while` loop **always runs at least once**, even if the condition is false.

**Syntax**

```c
do {
    // loop body
} while (condition);
```

ðŸ‘‰ Notice the **semicolon (;)** after `while(condition);` â€” many beginners forget it.

## 2) Print 1 to 5 (iteration trace)

```c
#include <stdio.h>
int main() {
    int i = 1;

    do {
        // Iteration 1: i=1 â†’ print 1 â†’ i++ â†’ i=2 â†’ check (i<=5) â†’ true
        // Iteration 2: i=2 â†’ print 2 â†’ i++ â†’ i=3 â†’ check (i<=5) â†’ true
        // Iteration 3: i=3 â†’ print 3 â†’ i++ â†’ i=4 â†’ check (i<=5) â†’ true
        // Iteration 4: i=4 â†’ print 4 â†’ i++ â†’ i=5 â†’ check (i<=5) â†’ true
        // Iteration 5: i=5 â†’ print 5 â†’ i++ â†’ i=6 â†’ check (i<=5) â†’ false â†’ stop
        printf("%d ", i);
        i++;
    } while (i <= 5);

    return 0;
}
```

**Output**

```
1 2 3 4 5
```

## 3) Reverse loop (5 to 1)

```c
int i = 5;
do {
    // Iteration 1: i=5 â†’ print 5 â†’ i--=4 â†’ check (i>=1) â†’ true
    // Iteration 2: i=4 â†’ print 4 â†’ i--=3 â†’ check (i>=1) â†’ true
    // Iteration 3: i=3 â†’ print 3 â†’ i--=2 â†’ check (i>=1) â†’ true
    // Iteration 4: i=2 â†’ print 2 â†’ i--=1 â†’ check (i>=1) â†’ true
    // Iteration 5: i=1 â†’ print 1 â†’ i--=0 â†’ check (i>=1) â†’ false â†’ stop
    printf("%d ", i);
    i--;
} while (i >= 1);
```

**Output**

```
5 4 3 2 1
```

## 4) Sum of first N numbers

```c
int N = 5, sum = 0, i = 1;

do {
    sum += i;
    // Iteration 1: i=1, sum=0+1=1 â†’ i=2
    // Iteration 2: i=2, sum=1+2=3 â†’ i=3
    // Iteration 3: i=3, sum=3+3=6 â†’ i=4
    // Iteration 4: i=4, sum=6+4=10 â†’ i=5
    // Iteration 5: i=5, sum=10+5=15 â†’ i=6 â†’ check fails
    i++;
} while (i <= N);

printf("Sum = %d\n", sum);
```

**Output**

```
Sum = 15
```

## 5) Factorial of a number

```c
int n = 5, fact = 1, i = 1;

do {
    fact *= i;
    // Iteration 1: i=1, fact=1*1=1 â†’ i=2
    // Iteration 2: i=2, fact=1*2=2 â†’ i=3
    // Iteration 3: i=3, fact=2*3=6 â†’ i=4
    // Iteration 4: i=4, fact=6*4=24 â†’ i=5
    // Iteration 5: i=5, fact=24*5=120 â†’ i=6 â†’ stop
    i++;
} while (i <= n);

printf("Factorial = %d\n", fact);
```

**Output**

```
Factorial = 120
```

## 6) Input until user enters 0

This is a **classic case** where `do-while` is better than `while`, since we want to **ask at least once**.

```c
#include <stdio.h>
int main() {
    int num, sum = 0;

    do {
        printf("Enter a number (0 to stop): ");
        scanf("%d", &num);
        sum += num;
        printf("Current sum = %d\n", sum);
    } while (num != 0);

    printf("Final sum = %d\n", sum);
    return 0;
}
```

**Sample Run**

```
Enter a number (0 to stop): 4
Current sum = 4
Enter a number (0 to stop): 7
Current sum = 11
Enter a number (0 to stop): 2
Current sum = 13
Enter a number (0 to stop): 0
Current sum = 13
Final sum = 13
```

## 7) Common Mistakes in `do-while` loops

1. **Forgetting the semicolon**

```c

   printf("Hello");
} while (i < 5)   // âŒ Missing semicolon
```

âœ… Correct:

```c
} while (i < 5);
```

---

2. **Infinite loop**

```c
int i = 1;
do {
    printf("%d ", i);
    // forgot i++ â†’ loop never ends âŒ
} while (i <= 5);
```

---

3. **Condition false, but still executes once**

```c
int i = 10;
do {
    printf("%d\n", i);
} while (i < 5);
```

**Output**

```
10
```

### âœ… Key Takeaways

* `while` checks **before** the loop; `do-while` checks **after**.
* `do-while` **always runs at least once**.
* Best for **menus, input validation, retry systems**.
* Donâ€™t forget the **semicolon** at the end.
