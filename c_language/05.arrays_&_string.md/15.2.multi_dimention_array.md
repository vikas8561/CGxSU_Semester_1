## 1) What is a Multi-Dimensional Array?

So far, we saw **1D arrays** (like a row of lockers).
But sometimes data is **tabular** (rows Ã— columns).

Examples:

* Student marks across subjects (2D grid).
* Pixel colors in an image (2D or 3D grid).
* Chess board (8Ã—8 squares).

ðŸ‘‰ A **multi-dimensional array** stores data in a **grid (or cube)** structure.
The most common is **2D array (matrix)**.

---

## 2) Declaring a 2D Array

**Syntax**

```c
datatype arrayName[rows][cols];
```

Examples:

```c
int marks[3][4];   // 3 students Ã— 4 subjects
float mat[2][2];   // 2Ã—2 matrix
char grid[5][5];   // 5Ã—5 character grid
```

---

## 3) Initializing a 2D Array

### A. Row-wise Initialization

```c
int mat[2][3] = {
    {1, 2, 3},     // row 0
    {4, 5, 6}      // row 1
};
```

### B. Flattened Initialization

```c
int mat[2][3] = {1, 2, 3, 4, 5, 6};
```

### C. Partial Initialization

```c
int mat[2][3] = {{1}, {4}};   // remaining values â†’ 0
```

---

## 4) Accessing Elements

Use **two indices**: `row` and `col`.

```c
printf("%d", mat[0][2]); // row 0, col 2 â†’ 3
printf("%d", mat[1][1]); // row 1, col 1 â†’ 5
```

---

## 5) Iterating Over a Matrix

```c
#include <stdio.h>
int main() {
    int mat[2][3] = { {1, 2, 3}, {4, 5, 6} };

    for (int r = 0; r < 2; r++) {
        for (int c = 0; c < 3; c++) {
            // Iteration trace:
            // r=0 â†’ c=0,1,2
            // r=1 â†’ c=0,1,2
            printf("%d ", mat[r][c]);
        }
        printf("\n"); // move to next row
    }
    return 0;
}
```

**Output**

```
1 2 3
4 5 6
```

## 6) Input and Output for 2D Arrays

```c
#include <stdio.h>
int main() {
    int marks[2][3];  // 2 students, 3 subjects

    printf("Enter marks for 2 students (3 subjects each):\n");
    for (int r = 0; r < 2; r++) {
        for (int c = 0; c < 3; c++) {
            scanf("%d", &marks[r][c]);
        }
    }

    printf("\nMarks table:\n");
    for (int r = 0; r < 2; r++) {
        for (int c = 0; c < 3; c++) {
            printf("%3d ", marks[r][c]);
        }
        printf("\n");
    }
    return 0;
}
```

**Sample Run**

```
Enter marks for 2 students (3 subjects each):
85 90 78
92 88 81

Marks table:

 92  88  81
```

## 7) Applications of 2D Arrays

### A. Sum of Elements in Matrix

```c
int mat[2][3] = { {1, 2, 3}, {4, 5, 6} };
int sum = 0;

for (int r = 0; r < 2; r++) {
    for (int c = 0; c < 3; c++) {
        sum += mat[r][c];
    }
}
printf("Sum = %d\n", sum);
```

**Output**

```
Sum = 21
```

---

### B. Row-wise and Column-wise Sum

```c
int mat[2][3] = { {1, 2, 3}, {4, 5, 6} };

for (int r = 0; r < 2; r++) {
    int rowSum = 0;
    for (int c = 0; c < 3; c++) rowSum += mat[r][c];
    printf("Row %d sum = %d\n", r, rowSum);
}

for (int c = 0; c < 3; c++) {
    int colSum = 0;
    for (int r = 0; r < 2; r++) colSum += mat[r][c];
    printf("Col %d sum = %d\n", c, colSum);
}
```

**Output**

```
Row 0 sum = 6
Row 1 sum = 15
Col 0 sum = 5
Col 1 sum = 7
Col 2 sum = 9
```

---

### C. Matrix Addition

```c
int A[2][2] = {{1, 2}, {3, 4}};
int B[2][2] = {{5, 6}, {7, 8}};
int C[2][2];

for (int r = 0; r < 2; r++) {
    for (int c = 0; c < 2; c++) {
        C[r][c] = A[r][c] + B[r][c];
        printf("%d ", C[r][c]);
    }
    printf("\n");
}
```

**Output**

```
6 8
10 12
```

---

### D. Transpose of Matrix

```c
int A[2][3] = {{1,2,3},{4,5,6}};
int T[3][2];

for (int r = 0; r < 2; r++) {
    for (int c = 0; c < 3; c++) {
        T[c][r] = A[r][c];  // swap row/col
    }
}

for (int r = 0; r < 3; r++) {
    for (int c = 0; c < 2; c++) {
        printf("%d ", T[r][c]);
    }
    printf("\n");
}
```

**Output**

```
1 4
2 5
3 6
```

## 8) Common Mistakes in 2D Arrays

1. **Wrong indices**

```c
printf("%d", mat[2][1]);  // âŒ out of bounds (row 2 doesnâ€™t exist)
```

2. **Confusing rows and cols**

```c
mat[1][2] vs mat[2][1]  // different elements
```

3. **Forgetting nested loops**

```c
for (int i = 0; i < 2; i++)
    printf("%d ", mat[i]);  // âŒ prints address, not elements
```

## 10) Key Takeaways

* **2D arrays** = grid-like structures (rows Ã— columns).
* Use **nested loops** for traversal.
* Applications: marksheets, images, tables, matrices.
* Be careful with **indices** (start at 0).
* Operations: sum, transpose, addition, searching.
